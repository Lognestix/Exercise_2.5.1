### Установка и запуск
1. Склонировать проект на свой компьютер
	- открыть терминал
	- ввести команду 
		```
		git clone https://github.com/Lognestix/Exercise_2.5.1
		```
1. Открыть склонированный проект в Intellij IDEA
1. Запустить авто-тесты:
	- Нажать два раза CTRL;
	- Ввести команду 
		```
		mvn clean test
		```
